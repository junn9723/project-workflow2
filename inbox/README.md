# inbox/ - ユーザー初期仕様書 受付ディレクトリ

> **ここに仕様書を置くだけで、PMが自動的にMVP開発を開始します。**

## 使い方

1. このディレクトリに仕様書ファイルを配置する（形式自由: .md / .txt / .pdf / 画像）
2. `/pm-start` を実行する
3. PMが inbox/ を検知し、仕様書を読み込み、MVP開発を開始する

## ファイル命名（推奨）
- `要件.md` / `requirements.md`
- `仕様.md` / `spec.md`
- `アイデア.md` / `idea.md`
- 任意の名前でOK（PMがすべて読み込みます）

## 処理後
- PMが仕様書を読み込み、正式な仕様書を `docs/specs/` に作成
- 処理済みの元ファイルは `inbox/processed/` に移動される
