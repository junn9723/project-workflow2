# タスク管理

> エージェント間連携の中核。全タスクはこのディレクトリにファイルとして管理する。
> **タスクファイルは自己完結的に記述し、どのメンバー（Claude Code / Codex CLI）でも単独で作業開始できるようにする。**

## ルール
1. タスクファイルは `TASK-XXX-description.md` の命名で作成
2. テンプレートは `.agent/templates/task.md` を使用
3. ステータス変更時は必ずタスクファイルを更新する
4. 差し戻し回数は正確に追跡する
5. **「実行メンバー」欄に claude_code / codex_cli を必ず明記する**
6. **「参照すべきファイル」欄に必要な全ファイルパスを列挙する**（Codex CLI の独立動作に必須）

## メンバー起動方法
| メンバー | 起動コマンド |
|---------|------------|
| Claude Code | `Task` ツール (サブエージェント) |
| Codex CLI | `./scripts/run-codex.sh .agent/tasks/TASK-XXX.md [Role]` |

## ステータスフロー
```
CREATED → IN_PROGRESS → SELF_REVIEW → IN_REVIEW → APPROVED → DONE
                                    → TESTING  → DONE
                                               → REJECTED (差し戻し)
```
