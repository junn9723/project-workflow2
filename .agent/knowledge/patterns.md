---

## 記録フォーマット

```markdown
### パターン名
- **カテゴリ**: architecture / design / implementation / test
- **問題**: どのような場面で使うか
- **解決策**: パターンの概要
- **実装例**: コードや設計の例
- **注意点**: 使用時の留意事項
- **検証**: どのタスクで検証されたか
```
