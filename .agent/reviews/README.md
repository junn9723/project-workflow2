# レビュー・テスト記録

> 設計レビュー、コードレビュー、テストレポートの記録。

## ルール
1. レビュー記録は `REVIEW-XXX-description.md` で作成
2. テストレポートは `TEST-XXX-description.md` で作成
3. テンプレートは `.agent/templates/review.md` / `.agent/templates/test-report.md` を使用
4. 知見があれば `.agent/knowledge/lessons-learned.md` にも転記
